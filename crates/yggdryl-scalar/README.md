# yggdryl-scalar

Arrow-centric **scalar values** for **yggdryl**: the growable in-memory `Binary`
buffer (which implements `yggdryl_core::Io`) and the validated `Utf8` string
value, behind the shared `Scalar` trait and the `AnyScalar` carrier.

Built on `yggdryl-core` and `yggdryl-dtype`. See `CLAUDE.md` at the repository
root for contributor rules.
