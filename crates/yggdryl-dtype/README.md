# yggdryl-dtype

Arrow-centric **data types** for **yggdryl**: the `DataType` trait hierarchy
(`PrimitiveType` / `NestedType` / `LogicalType`, plus `BinaryBased`) and the
concrete `BinaryType` / `Utf8Type` descriptors carried by the `AnyType` enum.

Built on `yggdryl-core`. See `CLAUDE.md` at the repository root for contributor
rules.
