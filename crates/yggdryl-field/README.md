# yggdryl-field

Arrow-centric **fields** for **yggdryl**: the generic `Field` (a named, nullable,
typed column with string metadata), its type-erased `AnyField`, and the
`PrimitiveField` / `NestedField` / `LogicalField` category markers.

Built on `yggdryl-core` and `yggdryl-dtype`. See `CLAUDE.md` at the repository
root for contributor rules.
